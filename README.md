# Pokedex with SWR

## Stack

- TypeScript
- Next.js
- SWR
- Tailwind
- Vercel

## Setup

Install the dependencies

```bash
$ yarn install
```

Then run the application:

```bash
$ yarn dev
```

Now the app will be running in http://localhost:3000
